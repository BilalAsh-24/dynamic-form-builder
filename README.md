# Dynamic Form Builder & Response Management System

## Overview
A full-stack MERN application that allows users to create custom forms, share them, and manage responses efficiently.

## Features
- User Authentication
- Dynamic Form Creation
- CRUD Operations
- Response Collection
- Form Sharing
- Responsive User Interface

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- MySQL

## Installation

1. Clone the repository
2. Install dependencies using:
   npm install
3. Start frontend and backend servers
4. Configure environment variables if required
